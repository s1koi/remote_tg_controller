# PC Remote acess TG-Bot

Telegram-бот для удалённого управления компьютером на Windows прямо из телефона.  
Скриншоты, управление мышью, файлы, процессы, диспетчер задач — всё через чат.

---

## Требования

- Windows 10 / 11
- Python 3.10+
- Telegram-аккаунт
- для нынешних реалий еще и прокси/VPN/VPS

---

## Установка
### Клонирование гитхаба
```bash
git clone https://github.com/s1koi/remote_tg_controller
```
### Установка библиотек для python
```
pip install pytelegrambotapi requests opencv-python Pillow mouse pySmartDL psutil
```
### Настройка конфига в "key.py"
```python
BOT_TOKEN = "TOKEN" # токен бота от @botfather
MY_ID = TG-ID # ваш телеграмм id 
```
### Запуск
```bash
python main1.py
```

## Структура Бота
```
Remote bot
├──main1.py  # фронтенд бота: хэндлеры, логика взаимодействия
├──logi.py   # Логика взаимодействия с пк
├──key.py    # местный config.py вместе с настройками клавиотур
├──logs.db   # логи бота для отладки
└──README.md
```

## Features/функции
### Скриншоты
| Кнопка | Описание |
|--------|----------|
| 📷 Быстрый скриншот | Скриншот со сжатием (фото) |
| 🖼 Полный скриншот | Скриншот без сжатия (документ) |
| 📹 Фото вебкамеры | Снимок с подключённой веб-камеры |

### Взаимодействия с файлами
| Кнопка | Описание |
|--------|----------|
| ✔️ Запустить | Открыть файл или программу по пути |
| ❌ Завершить процесс | Завершить процесс по имени (например: `chrome.exe`) |
| ⬇️ Скачать файл | Получить файл с ПК в Telegram |
| ⬆️ Загрузить файл | Отправить файл из Telegram на ПК |
| 🔗 Загрузить по ссылке | Скачать файл на ПК по прямой URL-ссылке |

### Дополнительные действия
| Кнопка | Описание |
|--------|----------|
| 🔗 Переход по ссылке | Открыть файл или программу по пути |
| ✔️ Выполнение команды | Выполнение команды через cmd |
| ⛔️ Выключить компьютер | Выключить ПК |
| ♻️ Перезагрузить компьютер | перезагрузка ПК |
| 🖥 О компьютере | Имя пользователя, IP, ОС, процессор |

### Диспетчер задач
| Кнопка | Описание |
|--------|----------|
| 📋 Все процессы | Открыть файл или программу по пути |
| 🔥 Топ по CPU | 10 процессов по нагрузке CPU |
| 🔥 Топ по GPU | 10 процессов по нагрузке GPU |
| 💾 Топ по RAM | 10 процессов по нагрузке RAM |
| 📈 Состояние системы | загрузка всех важных компонентов |
| 🔍 Найти процесс | Найти процесс по части имени |
| ❌ Завершить по pid | Завершить процесс по pid |

### Логи бота

Записывание всех процессов бота в удобные логи .db

### Система безопасности

Бот реагирует только на сообщения от id пользователя, увазанного в `key.py`
Все другие запросы блокируются и записываются в логи

### Всплывающее уведомление

Можно вызвать всплывающее окно-уведомление на пк с указанным текстом

### Управление мышью

Взаимодействие с мышью реализовано через inline клавиотуру
Можно настраивать размах мыши

## Используемые библиотеки

| Библиотека | Описание |
|------------|------------|
| `Telebot` | Библиотека взаимодействия с тг |
| `requests` | запросы в браузер и работа с ссылками |
| `opencv` | Взаимодействие с изображениями и вебкамерой |
| `pillow` | Скриншоты и поиск курсора |
| `mouse` | управление мышью |
| `psutil` | Имитация диспетчера задач |
| `sqlite` | Работа логов через bd |
